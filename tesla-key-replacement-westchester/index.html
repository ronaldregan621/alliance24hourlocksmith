Tesla Key Types and What We Can Help With

Key Card

Model 3 and Model Y use an RFID key card as the backup access method. The card taps the B-pillar or center console to unlock and start the car. If the card is lost, a replacement must be ordered through the Tesla app and then paired via the vehicle touchscreen.

Locksmith: Lockout help from $120

Phone Key

Most Tesla owners use their phone as the primary key via Bluetooth. If the phone battery dies or the app loses pairing, access is lost. We help you regain entry and can assist with getting you to a charging source or backup key activation.

Locksmith: Lockout assistance from $120

Model S and X Fob

Older Model S and Model X vehicles (pre-2021 refresh) use a traditional push-button fob shaped like a miniature Tesla. Replacement fobs are ordered from Tesla and must be paired via the vehicle touchscreen while you have at least one working key.

Locksmith: Replacement guidance, lockout from $120

If you are physically locked out of your Tesla with no access method working, we can reach the interior using professional non-destructive tools. We do not manufacture Tesla replacement keys, but we get you back inside and advise on the fastest path to a new access credential.

Tesla Lockout and Key Questions

Can a locksmith make a replacement Tesla key?

No, and anyone who claims otherwise is not being accurate. Tesla key cards and phone key credentials are managed through Tesla's proprietary system and must be paired via the vehicle's touchscreen while you have at least one valid access credential. What we can do is get you back inside the car physically using non-destructive entry, so you can then activate a replacement card or reconnect your phone.

How do I get back into my Tesla if my phone is dead and I lost my key card?

Call us first. We can open the car using non-destructive tools. Once inside, if the car has any charge, you can use the touchscreen to contact Tesla Support or use the Tesla app on any device to add a new key card. We can also tow you to the nearest Tesla Service Center if needed.
